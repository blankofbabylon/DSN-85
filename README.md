# DSN-85
Here's a short description for the easily changeable elements in my theme (in html{})
Sooner or later, I'll make a readable css file that simply imports from this place and only has these values.
<br>
<br>
The hue of your color: --wlm-topbar-accent-colorH <br>
The saturation of your color: --wlm-topbar-accent-colorS <br>
The main luminance of your color: --wlm-topbar-accent-colorL <br>
The secondary luminance of your color (used for lighter elements): --wlm-topbar-accent-colorLlight <br>
The tertiary luminance of your color (only used in user section): --wlm-topbar-accent-colorLtopbar <br>
<br>
Height of caption bar: --wlm-topbar-tophalf-height <br>
Height of user section: --wlm-topbar-bothalf-height <br>

Font for most elements: --wlm-system-font-family <br>
Size for most fonts: --wlm-system-font-size <br>
Font for messages: --wlm-chat-font-family <br>
Size for message fonts: --wlm-chat-font-size <br>

Radius of the top corners, change to 0px if native titlebar is used: --wlm-topcorner-radius <br>

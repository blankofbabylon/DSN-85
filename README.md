# DSN-85
Here's a short description for the easily changeable elements in my theme (in html{})
Sooner or later, I'll make a readable css file that simply imports from this place and only has these values.

The hue of your color: --wlm-topbar-accent-colorH
The saturation of your color: --wlm-topbar-accent-colorS
The main luminance of your color: --wlm-topbar-accent-colorL
The secondary luminance of your color (used for lighter elements): --wlm-topbar-accent-colorLlight
The tertiary luminance of your color (only used in user section): --wlm-topbar-accent-colorLtopbar

Height of caption bar: --wlm-topbar-tophalf-height
Height of user section: --wlm-topbar-bothalf-height

Font for most elements: --wlm-system-font-family
Size for most fonts: --wlm-system-font-size
Font for messages: --wlm-chat-font-family
Size for message fonts: --wlm-chat-font-size

Radius of the top corners, change to 0px if native titlebar is used: --wlm-topcorner-radius
